# Manesis Auto Service — Οπτική Ταυτότητα

Παραδοτέα brand identity για το συνεργείο **Manesis Auto Service** (Σπάτα).

## Περιεχόμενα

- `index.html` — αρχική σελίδα με links σε όλα τα παραδοτέα
- `presentation.html` — slideshow παρουσίασης (πλοήγηση με βέλη)
- `logos.html` — 3 κατευθύνσεις λογοτύπου (CALIPER, BLUEPRINT, REDLINE)
- `cards.html` — εκδοχές επαγγελματικής κάρτας (μπροστά/πίσω)
- `website.html` — μονοσέλιδο website (desktop)
- `website-mobile.html` — mobile εκδοχή website

## Ανέβασμα στο GitHub Pages

1. Δημιουργήστε νέο repository και ανεβάστε **όλα** τα αρχεία του φακέλου (μαζί με `support.js`, `deck-stage.js`, `image-slot.js` και τον φάκελο `export/`).
2. Settings → Pages → Source: **Deploy from a branch** → branch `main`, folder `/ (root)`.
3. Το site θα είναι διαθέσιμο στο `https://<username>.github.io/<repo>/`.

Όλα τα αρχεία φορτώνουν με σχετικές διαδρομές — δεν χρειάζεται καμία ρύθμιση.
